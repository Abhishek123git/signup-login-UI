## 🛠 Getting Started

Follow these steps to run the project locally:

### 1. Clone the repository
```powershell
git clone https://github.com/Abhishek123git/signup-login-UI
```

### 2. Go to the project directory
```powershell
cd signup-login-UI
```

### 3. Install dependencies
```powershell
npm install
```

### 4. Start the server
```powershell
npm start
```
